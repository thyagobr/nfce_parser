# nfce parser

1. Baixe o HTML no site descrito em qualquer nota fiscal
2. Coloque o arquivo no root do script como nome "nfce.html"
3. `ruby nfce_parser.rb`

URL: http://nfce.set.rn.gov.br/portalDFE/NFCe/ConsultaNFCe.aspx
